# M'mawa Seed Solutions Ltd — Image Assets

This directory is reserved for image assets.

## Current Status

The website currently uses CSS gradients, inline SVG icons, and decorative shapes instead of external image files. This ensures the site works without any image dependencies.

## Future Image Assets

When ready, add the following images:

- `logo.png` — Company logo
- `hero-bg.jpg` — Hero background (tobacco field at dawn)
- `tobacco-field.jpg` — Tobacco field photography
- `seed-pack.jpg` — Certified seed packaging
- `farmer-support.jpg` — Farmer support imagery
- `supply-route.jpg` — Zimbabwe to Malawi supply route map
- `batch-verification.jpg` — Batch verification process

## Image Guidelines

- Use high-quality, authentic photography from Malawi and Zimbabwe
- Avoid generic stock photos
- Ensure images reflect real tobacco agriculture
- Optimise all images for web (compress, use appropriate formats)
- Recommended hero image size: 1920x1080px
- Recommended card images: 800x600px
